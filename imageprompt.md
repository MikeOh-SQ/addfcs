# ASRS / DSM-5 Image Prompts

서비스에서 교체할 문항별 이미지 생성 프롬프트 문서입니다.

대상 파일:
- `public/asrs01.png` ~ `public/asrs06.png`
- `dsmimages/d1.png` ~ `dsmimages/d23.png`

## Core Rule

- 기본 캐릭터는 항상 곰입니다.
- 토끼는 꼭 필요한 상호작용 장면에서만 보조 캐릭터로 사용합니다.
- 토끼가 등장해도 주인공은 항상 곰이어야 합니다.

```text

```

## Shared Output Rules

```text
the bear must remain the clear main character, optional rabbit companion only if interaction is necessary, no text, no speech bubble, no props covering the face, clear readable pose, emotionally clear storytelling pose, front or 3/4 view, plain solid white background, no shadow on background, no border, no watermark
```

## Negative Prompt

```text
multiple main characters, realistic photo, 3d render, dark mood, hospital atmosphere, busy background, furniture-heavy scene, readable text, caption, logo, watermark, cropped body, close-up portrait only, extra limbs, distorted hands, scary expression
```
